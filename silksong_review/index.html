[心得] 絲之歌，令人沉浸與感動的神作，破關感想

📅 Oct 1, 2025  ·  ☕ 9 min read
    🏷️
  • #心得
  • #電玩
  • #switch
  • #indie
  • #metroidvania

2019 揭露後過了六年，總算是玩到了心心念念的空洞騎士續作絲之歌，連續玩了二十多天，真結局破關，覺得超級好玩、超級沉浸，遊戲整體超機巴難，但是結局也是讓人超級感動。

這是款 2D 動作冒險遊戲，有著既艱難又暢快的戰鬥體驗，以及極致沉浸的探索氛圍；前作已經是完成度很高的神作，而這部絲之歌，神作的續作還是神作！以下就介紹一下作品特色並夾雜個人遊玩感想。

文章對劇情與遊玩內容都有所揭露，斟酌觀看。

蟲與絲

空洞騎士是一個以各種昆蟲或是類似生物為基礎，將他們擬人化的遊戲，在前作用了蟲子趨光特性講述精神被"輻光"飛蛾感染的故事，這次則是從絲線發揮。在一個稱為紡絡的地帶，蟲民們建立了使用機械的高度文明，甚至發展出蟲機一體的技術，但在某天來了位帶絲后蟲，紡絡王國從此變調。

現實中，昆蟲一旦纏上蜘蛛絲就難以掙脫，遊戲將這現象轉換為對絲的迷茫嚮往；舉國上下開始崇拜女王，對絲線的執著化為一種詛咒，讓文明從此由盛轉衰，終至孤寂；大部份蟲民皆變成沒有靈魂、被絲線所控制的的行屍走"蟲"。

主角大黃蜂因為自身的能力，被女王派人抓來此地；玩家就跟著她一路探尋此地秘密，並嘗試打破這絲之信仰背後的詛咒。本以為收拾了女王就可天下太平，想不到薩滿鍋牛給的餿主意，以虛無封印女王，反而弄巧成拙，讓虛無透過絲線滲透到了紡絡，國度崩壞，實質意義上的"黑慘了"XD。劇情沒有什麼大起大落與轉折，但做為動作冒險遊戲的故事元素，已很引人入勝。

使針好手

大黃蜂以武器紋章、縛絲技能、投擲道具，三種東西互相組合來戰鬥。雖然拿了根針，但透過紋章形式，就能使出類似其他武器的攻擊方式和手感；輕盈劍、厚重斧、俐落短刀、豪邁爪(覺得很難用)等等，比舞棍悟空還行。

輕盈的刺客，遊戲的打怪手感極好。

紋章有插槽，可裝一個縛絲技能+其餘類別的裝飾，裝飾物的格子數有限，後續能透過道具有限地擴充容量；紅插槽用來裝投擲的遠程道具或是陷阱，做為副武器輸出；藍、黃插槽則偏戰鬥以外的輔助功能。

打怪戰鬥做為這款動作遊戲最直接的玩點，有別於一代的體驗，藉由紋章、技能、道具與飾品之間的組合，看自身需求與習慣去調整和取捨，變化是更多。我個人並不熱衷於 Build 配裝元素，從頭到尾都是用初始獵手針，後面陸續套上快速縛絲、加長範圍的長爪，沒什麼更換嘗試；東西是撿爽的，而不是拿來用XD。

沉浸探索

對我來說，空洞騎士最吸引人的地方莫過於優美的場景，配上悠悠淒涼的音樂與若有似無的擺設細節，關卡設計的氛圍感極佳。地圖可簡略分成原野、聖堡和夢境區。依照取得的移動能力，各區塊有著明顯的解鎖順序，但又兼具一定程度的開放探索，闖關的節奏和推進感很不錯；沒有一口氣攤開太多區域，讓人無所適從。

地圖路線交錯蔓延，時而藏有隱密空間，在不斷迷路和找路的過程中，逐漸揭露地圖全貌，終至豁然開朗；即使是 2D 平台類型，憑著優異的氛圍和區域主題，找路探索超級沉浸與享受。

遊戲很難，雜魚和陷阱都多，要找到椅子才能回血存檔，死亡後則會從上回的座椅處開始；所以每次到了新區域，越是深入險地，越是步步為營；最幹的就是走了一大段，還沒坐到椅子就葛屁，那就是抱歉請重走。因為關卡美術的優秀，再加上小心謹慎不想死的冒險感，就讓人對探索過程很投入。

景致該有的都有：翠綠清新的苔地、下雨孤寂的沼澤、寒風刺骨的尖山、黃沙侵蝕的階地等等區域，整體偏暗，但是搭配不同色調，視覺體驗很優秀。

最令我驚艷的是第二幕中的聖堡，這座城堡內有多種設施與場所，書庫、議會廳、餐廳、溫泉、劇場、植物館、實驗廳，彼此之間穿插著眾多廊室。城堡中央是由機械齒輪構成的高塔，左側是入城必經的台階高地，右後方有水渠街道；城堡底下則是有著採礦研磨、蒸汽鎔鑄的一整座巨大工廠；以城堡主題延伸出的區域關卡，什麼小都有。

在溫泉處碰到熟悉的 NPC，一起泡一下，倍感溫馨。

在打倒門扉前機巴強的守衛後，一踏入城區，響起了空靈神聖的音樂，真的讓人有種朝聖晉見的感覺；高聳宏偉的城堡是代代蟲子們努力的成果，如今徒留傾頹衰敗的光景，訴說著昔日曾有的輝煌；遊戲世界在製作組的精心打造下，真是令人悠然神往。(超神BGM )

後期夢境圖，失落的翠庭我也很喜歡；翠綠的庭園，相比黑壓壓的崩壞紡絡，綠意盎然+清澈碧池，極舒服的一個區域。小怪都是綠色的螳螂螽斯，還有一隻像是神明的麋鹿，整個很吉卜力魔法公主XD。

聲音元素

絲之歌有著不少關於聲音方面的元素。大黃蜂的針穿了線能變成弦樂器，以彈奏方式觸發事件、開啟機關、伴奏讓 NPC 甚至是敵人哼起歌；這讓我想起老任的幾款薩爾達舊作，也是將樂器演奏融入了遊玩機制當中。

聖堡全名聖歌城堡，以宗教旋律歌頌聖詠這個國家與主宰；大黃蜂取得三段重要旋律後，在雕像面前來了個宏大高亢的合唱演奏。

其中一種收集物聖詠音筒也很有意思，把找來的唱帶交給管理員，可以用一旁的留聲機播出宗教味道濃厚又神聖的聖詠曲目。

在無止盡下雨的灰沼，Switch2 手把會配合畫面，透過震動馬達的淫威，發出滴滴嘟嘟的雨聲；為了臨場氛圍，對於細節的打磨，櫻桃真是走火入魔。

鮮活角色

不論是敵人還是友方 NPC，都有著簡單語音和言行，讓人覺得很鮮活；跳蚤旅團、送貨甲蟲、磨針工匠、跑步選手等等都很有特色。

尤其喜歡傑爾瑪、加蒙德、努努這三位NPC。小個兒傑爾瑪，虔誠又天真的朝聖者，初期看他敲著鈴鐺想要打開路上的門，憨得可愛。之後每次相遇，都能察覺到她想法上的轉變，信仰動搖、困惑疲憊、失落沮喪；之後打起精神、鼓舞助人、引領眾蟲；在最終戰之前，還會唱歌祝福大黃蜂。

加蒙德是個甲蟲騎士，在打超強蒼蠅時，長叫一聲 “引~突巴” 豪邁登場，時不時高呼 “Sexy Bone~"，非常逗趣。

後續在其他地圖也會看到大叔與他的座騎夥伴奮勇殺怪；最後卻被虛空感染，為了解脫正義樂觀的老朋友，只能幹掉他，他的坐騎會陪背著主人的遺體；蠻難過的，這麼討喜的 NPC 卻是輓歌定位的洋蔥騎士，彈奏一曲，為他送行。

努努長得像蝸牛，但身後其實是背包而不是殼；本人喜好研究各種怪物，更是那方面的老饕，跟任天堂皮克敏中的路易一樣敢吃(有板友聽過嗎?)。平常待在酒館中，後續會在幾個玩家擊敗Boss的屍骸處看到他出沒，研究食材與吃法XD。

敵人在攻擊、挨打與翻肚死去的簡單配音也很不錯，像這隻初期就會遇到，在地板匍匐前行、中了絲咒的朝聖者，會用猥褻的聲音撲過來，很搞笑。

小怪的分配也會跟區域情境有所關聯。在聖堡工廠，拿著抹布跟雞毛撢子在打掃的小怪，做著不折不扣的雜魚活兒XD，看到笑出來；圖鑑說明，大黃蜂還很正經的評論：「清潔工具與周圍同樣汙穢，這樣打掃只會越來越髒」有趣。

要你命 Boss

除了探索之外，遊戲另一大玩點就是擺在豐富的 Boss 戰，牠們都有著獨特的外型與攻擊模式：齒輪驅動的兩隻機器蟲，用機械式的舞步戰鬥；編織絲線，跑來跑去、手超忙的黑寡婦；頭尾都會攻擊，有點噁心的大蜈蚣；喜歡搞蟑螂菜餚而被聖堡趕走、會用大屁屁頂撞的超肥廚子。

特別喜歡末代裁決者和卡梅莉塔。末代裁決者，站在聖堡門口的守衛，要進入就得經過他無情的審判：不潔的蟲子，死！鐵面無私的撲克臉，配上金色光閃閃的盔甲，揮著炙熱冒煙的流星錘，大開大闔，霸氣十足。最後用正義織針推翻了裁決，去你媽的裁決。

斯卡爾歌后，卡梅莉塔，蟻族的首領，戰鬥場所是劇台，背景還有諸多螞蟻觀眾。整個過程，她用渾厚嗓音唱著歌劇，手上快速轉著雙刃刀，以華麗的曼妙舞步攻擊，快速又致命；超幹難打，總共試了兩個半小時才贏，雖難但是印象很深刻。

雜七雜八

支線任務的導入還不錯，明確給出任務目標，在跑圖探索之餘，調節一下遊玩節奏。

鐘居，屬於主角的小套房，能花錢擺設一些家具，沒什麼實質作用，但就是爽；如同 DQ 的教會，冒險過後的落腳處，讓人很有歸屬感，有事沒事就會去一下，而且小鎮的音樂極療癒，真的有放鬆身心靈的感覺。(超神BGM )

Boss 打人超超痛血又厚，部分還需要先車輪戰，或是叫上雜魚搗亂，或是重生點超遠，這些都很機八；自己主要遊玩任系遊戲，平台跳躍的作品玩了很多，但是絲之歌的平台跳躍內容，我覺得也很有挑戰性。遊戲真的很難，但是關關難過關關過，隨之而來的成就感，令人難忘。

黃蜂女俠

我自己有卡關的部分是在第三幕的內容，幾個頭目都讓我苦戰許久，但是真結局的過場演出讓我覺得苦盡甘來，非常值得。

在前作，大黃蜂看著故鄉聖巢的衰敗，她無能為力；而這次對於被黑暗虛無滲透、危在旦夕的紡絡，與各處 NPC 的對話，都是要大家別放棄、保護好自己，她絕對會拯救此地。為了取得關鍵能力，會先去一次深淵，再從爆發的岩漿地帶返回地面，極熱血。

一人一針如同女俠，上達聖堡下至深淵，進入夢境之中，挑戰並獲取強者之心，焚燒並注入自己的身體，讓自己回到久遠的記憶片段中，拿到那朵可以對抗黑暗、永恆綻放的小白花(有那麼一點全面啟動的感覺XD)。

最後幾段演出，串起了一代的內容：大黃蜂是蒼白國王沃姆與織者的血脈，通過各種磨練，承載了母親、師傅與王后的期望，並有著自身的信念。或許大黃蜂已經活得太久、見過太多，心態麻木了，回到最初取得花朵也有那種找回初始信念的意涵。

萬事俱備後，跳進深淵的核心，直面罪惡源頭。在黑暗之中，大黃蜂拿著小花，透出一丁點光亮，持續下沉；小花讓其免於黑暗的吞噬，同時也象徵著她就是紡絡的希望，配上磅礡弦樂，整個超感動！覺得遊戲中各種艱難的苦戰都值了。

打敗最後敵人要跳離深淵時，小花撐不住謝了，即將被虛無包覆前，前作的主角小騎士，救了大黃蜂一把；前作大黃蜂幫助小騎士，這次換小騎士出手，兩作完美的聯接在一起。

結語

絲之歌繼續發揚光大一代傑出的探索沉浸感與過癮的 Boss 挑戰，遊玩元素無敵豐富，關卡美術與主題性，還有配樂音效，都有著巨大進步，且絲毫沒有重複、缺少新鮮感這種直接續作難免會有的缺點；故事則是巧妙地銜接一代的設定，讓玩過前作的玩家(我)非常滿足，真心覺得神作。
